Configuração do Firebase
Siga os passos abaixo para conectar o banco de dados gratuitamente
⚠️ O site ainda está usando uma configuração de exemplo. Para que os dados sejam salvos de verdade, você precisa criar um projeto no Firebase e substituir as chaves no arquivo HTML.
  1. Acesse console.firebase.google.com e faça login com a conta Google da Ágora EJ.
  2. Clique em "Criar projeto", dê o nome agora-ej e prossiga (pode desativar o Google Analytics se quiser).
  3. Dentro do projeto, clique em "Firestore Database" no menu lateral, depois em "Criar banco de dados" → escolha Modo de teste (ajuste as regras depois).
  4. Volte à tela inicial do projeto e clique no ícone </> (Web) para registrar o app. Anote as credenciais exibidas.
  5. Abra o arquivo agora-ej.html em um editor de texto, encontre o bloco firebaseConfig e substitua pelos seus dados:
    apiKey: "SUA_API_KEY_AQUI", authDomain: "agora-ej.firebaseapp.com", projectId: "agora-ej", storageBucket: "agora-ej.appspot.com", messagingSenderId: "123456789", appId: "1:123:web:abc"
  6. Para colocar online, crie uma conta em github.com, suba o arquivo agora-ej.html em um repositório e ative o GitHub Pages nas configurações do repositório. O link fica disponível para todo o time da Ágora.
